## 📌 Project Overview

This project focuses on converting **unstructured government job notification PDFs** into a **structured dataset** using data analytics techniques.

📊 The goal was to:
- Extract key job-related information  
- Standardize data across multiple organizations  
- Ensure **accuracy, consistency, and usability**  

---

## 🧠 Problem Statement

Government job notifications are often:
- ❌ Unstructured  
- ❌ Inconsistent  
- ❌ Difficult to analyze  

👉 This project solves that by transforming raw data into a **clean tabular format**.
